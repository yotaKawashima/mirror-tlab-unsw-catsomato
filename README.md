# tlab-unsw-catsomato

This repository is for Kawashima Y, Li R, Chen SCY, Vickery RM, Morley JW, et al. (2021) Steady state evoked potential (SSEP) responses in the primary and secondary somatosensory cortices of anesthetized cats: Nonlinearity characterized by harmonic and intermodulation frequencies. PLOS ONE 16(3): e0240147. https://doi.org/10.1371/journal.pone.0240147. You can replicate the analysis with scripts here.

Note that data to be analysed are not included in the data directory. Please download the data at [] and put them in data/raw/ directory.

See [Home.md](./documentation/Home.md) for documentation.
